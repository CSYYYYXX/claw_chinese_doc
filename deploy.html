云服务器部署（VPS）

按官方文档思路整理：先跑通，再加固，再自动化。

🧭部署路线图

1) 选平台

官方推荐入口：Railway / Northflank / Fly.io / Hetzner / GCP / Oracle。总览：VPS Hosting

2) 安装并配置

bash
npm install -g openclaw@latest openclaw onboard

3) 守护运行

官方网关管理命令：

bash
openclaw gateway status openclaw gateway start openclaw gateway restart

🔒安全基线（强烈建议）

  • 默认仅监听本机回环（loopback），优先走 SSH 隧道或 Tailscale。
  • 如果需要外网访问，务必启用 gateway 认证（token/password）。
  • Bot Token / API Key 只放服务器端，不写进前端页面和公开仓库。
  • 使用最小权限原则：Discord/Slack 不要给 Administrator。

🛠️运维与排错

健康检查

openclaw status 查看模型、会话、网关状态。

日志定位

openclaw logs --follow 持续观察报错与重连行为。

自动修复

openclaw doctor 检查常见配置问题。
