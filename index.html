打造你的专属 AI 助手

OpenClaw 是一个可以在你自己的电脑上运行的 AI 助手。
它可以帮你处理消息、控制设备、自动化任务——而这一切都是私密的、本地的。

🚀 开始安装 📖 先看看教程

💡 OpenClaw 是什么？

简单来说，它就像一个住在你的电脑里的智能管家。

💬 多平台消息

连接 Telegram、Discord、Slack 等多个聊天平台，一个 AI 助手统一回复。

🔒 隐私安全

所有数据都在你自己的电脑上，不会上传到云端。你的隐私，你说了算。

🤖 智能自动化

可以帮你执行命令、管理文件、设置定时任务，甚至控制智能家居设备。

🔌 无限扩展

通过 Skill（技能）系统，可以添加新功能，比如天气查询、邮件发送、股票监控等。

⚡ 三步即可上手

即使你完全不懂编程，也能在 10 分钟内完成安装。

安装 Node.js

Node.js 是运行 OpenClaw 的基础环境。去 nodejs.org 下载安装包，一路"下一步"即可。

ℹ️
小贴士
选择 LTS（长期支持版）更稳定，安装时记得勾选"Add to PATH"选项。

一条命令安装 OpenClaw

打开命令行（Windows 按 Win+R 输入 cmd），复制粘贴这一行：

命令行
npm install -g openclaw@latest

启动配置向导

安装完成后，运行这条命令，跟着向导一步步操作：

命令行
openclaw onboard

向导会帮你完成所有配置，包括选择 AI 模型、设置消息渠道等。

✨ 它能帮你做什么？

📱 消息助手

在 Telegram/微信里随时提问，AI 帮你查资料、写文案、翻译、总结文章。

⏰ 定时提醒

设置定时任务，比如每天早上发天气预报、提醒开会、推送新闻摘要。

🌐 浏览器控制

让 AI 帮你打开网页、填表单、截图，自动化处理重复的网页操作。

📁 文件管理

搜索本地文件、整理文件夹、批量重命名，文件杂乱不用愁。

🏠 智能家居

通过语音或消息控制智能设备，"开灯"、"关空调"一句话搞定。

🔧 更多可能

通过 Skill 系统扩展功能，或者自己写一个专属技能。

✅ 环境基线

建议 Node.js 22+，首次配置后用 openclaw status --deep 和 openclaw doctor 做体检。

🔐 安全策略

生产环境启用强口令、限制来源 IP、通道默认 requireMention，避免 Bot 被公共群组滥用。

🛠️ 排错闭环

优先执行 openclaw status → openclaw logs --follow → openclaw doctor，再回看渠道权限与密钥状态。
