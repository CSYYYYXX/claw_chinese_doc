CLI 命令大全

把常用命令按场景分组，你只需要复制执行即可。

🚀安装与初始化

bash
npm install -g openclaw@latest openclaw --version openclaw onboard openclaw onboard --install-daemon

🧭状态与诊断

基础状态

openclaw status
查看运行态、模型、渠道概览。

深度状态

openclaw status --deep
显示更细粒度健康信息。

一键体检

openclaw doctor
排查依赖、权限、配置问题。

🔧网关生命周期

bash
openclaw gateway start openclaw gateway stop openclaw gateway restart openclaw gateway status

📜日志与调试

bash
openclaw logs --follow openclaw dashboard

排错推荐顺序：status → logs --follow → doctor。

💬会话与历史

bash
openclaw sessions list openclaw sessions export <session-key> > session.json

🧩排错模板

模板 1：渠道不回消息

检查权限与 mention 策略，日志中查 401/403/429。

模板 2：模型请求失败

确认 API Key、额度、地区网络代理、base URL。

模板 3：重启丢服务

确认执行过 onboard --install-daemon。
