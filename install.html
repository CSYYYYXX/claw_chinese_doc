安装教程

根据你的操作系统，选择对应的安装方式。整个过程大约需要 10 分钟。

🖥️ 选择你的操作系统

🪟 Windows

Windows 10/11，推荐使用 WSL2

🍎 macOS

支持 Intel 和 Apple Silicon

🐧 Linux

Ubuntu、Debian、CentOS 等

🪟 Windows 安装步骤

💡
推荐：使用 WSL2
WSL2 可以在 Windows 上运行 Linux 环境，兼容性更好。如果你想用纯 Windows，也可以，继续往下看。

安装 Node.js

1. 访问 https://nodejs.org

2. 下载 LTS 版本（长期支持版，更稳定）

3. 运行安装程序，一路点"下一步"

4. 安装完成后，按 Win + R，输入 cmd，回车

5. 输入以下命令验证安装：

命令行
node --version

如果显示版本号（如 v20.x.x），说明安装成功！

安装 OpenClaw

在命令行中执行：

命令行
npm install -g openclaw@latest

等待几秒钟，看到进度条走完就安装好了。

⚠️
如果报错
如果提示"npm 不是内部或外部命令"，说明 Node.js 没装好或没加到环境变量。重新安装 Node.js，确保勾选"Add to PATH"选项。

运行配置向导

输入这条命令启动向导：

命令行
openclaw onboard

向导会引导你：

  • 选择 AI 模型（推荐 OpenAI 或 Anthropic）
  • 配置消息渠道（Telegram、Discord 等）
  • 安装 Gateway 服务（让 AI 在后台持续运行）

启动 Gateway

配置完成后，启动网关服务：

命令行
openclaw gateway

看到 Gateway running on http://127.0.0.1:18789 就成功了！

打开浏览器访问 http://127.0.0.1:18789 可以看到控制面板。

🍎 macOS 安装步骤

安装 Homebrew（包管理器）

Homebrew 是 macOS 上最常用的包管理器，可以方便地安装软件。

打开"终端"（按 Cmd + 空格，输入"终端"），粘贴以下命令：

终端
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

按回车执行，等待安装完成。

安装 Node.js

终端
brew install node

验证安装：

终端
node --version

安装 OpenClaw

终端
npm install -g openclaw@latest

运行配置向导

终端
openclaw onboard

启动 Gateway

终端
openclaw gateway

或者安装为后台服务：

终端
openclaw onboard --install-daemon
✨
macOS 专属：菜单栏应用
OpenClaw 提供了 macOS 菜单栏应用，可以方便地查看状态、快速操作。在向导中选择安装即可。

🐧 Linux 安装步骤

安装 Node.js

Ubuntu/Debian：

bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - sudo apt-get install -y nodejs

CentOS/RHEL：

bash
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash - sudo yum install -y nodejs

安装 OpenClaw

bash
npm install -g openclaw@latest

运行配置向导

bash
openclaw onboard --install-daemon

--install-daemon 会将 OpenClaw 安装为系统服务，自动启动。

启动 Gateway

bash
openclaw gateway
💡
服务器部署
如果你想把 OpenClaw 部署到云服务器上，可以参考 VPS 部署指南。

✅ 验证安装

安装完成后，用这些命令检查是否正常工作。

命令行
openclaw status

如果看到类似下面的输出，说明一切正常：

输出示例
OpenClaw status Overview ┌─────────────────┬──────────────────────────────────────┐ │ Item │ Value │ ├─────────────────┼──────────────────────────────────────┤ │ Gateway │ running · ws://127.0.0.1:18789 │ │ Model │ openai-codex/gpt-5.3-codex │ │ Sessions │ 1 active │ └─────────────────┴──────────────────────────────────────┘
