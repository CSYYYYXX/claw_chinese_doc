Gateway & Dashboard

运行、监控与排错的核心章节。

⚙️生命周期命令

bash
openclaw gateway start openclaw gateway stop openclaw gateway restart openclaw gateway status

🖥️控制面板与日志

bash
openclaw dashboard openclaw logs --follow openclaw status --deep openclaw doctor

📌运行建议

  • 长期运行请使用 openclaw onboard --install-daemon。
  • 本地测试可前台运行：openclaw gateway --port 18789。
