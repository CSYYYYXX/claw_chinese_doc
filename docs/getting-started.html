Getting Started

目标：从 0 到第一次可用聊天。

✅前置条件

  • Node.js 22 或更新版本。
  • 可用的模型供应商 API Key。
  • 终端可执行 node -v 与 npm -v。

⚡最快路径（CLI）

1. 安装

bash
npm install -g openclaw@latest openclaw --version

2. 向导初始化

bash
openclaw onboard --install-daemon

3. 检查网关

bash
openclaw gateway status openclaw status --deep openclaw doctor

4. 打开控制台

bash
openclaw dashboard

默认地址通常是 http://127.0.0.1:18789/。
