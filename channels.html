📱 Telegram Bot 配置

5 分钟完成配置，最推荐新手使用。

创建 Bot

  1. 在 Telegram 搜索 @BotFather
  2. 发送 /newbot
  3. 按提示输入 Bot 名称（如 "我的AI助手"）
  4. 输入 Bot 用户名（必须以 bot 结尾，如 my_ai_assistant_bot）
  5. 复制 BotFather 返回的 Token（类似 123456789:ABCDefGHIjklMNOpqrsTUVwxyz）

配置 OpenClaw

运行配置向导：

命令行
openclaw onboard

选择 Telegram，粘贴刚才的 Token。

或者直接编辑配置文件：

~/.openclaw/openclaw.json
{ "channels": { "telegram": { "enabled": true, "botToken": "你的 Bot Token", "dmPolicy": "pairing", "groups": { "*": { "requireMention": true } } } } }

启动并测试

命令行
openclaw gateway

在 Telegram 找到你的 Bot，发送 /start 开始对话！

✨
Telegram 优势
响应速度快、支持 Markdown 格式、支持文件传输、多端同步、完全免费。
🛠️
Telegram 常见问题排查
收不到消息：先确认 Bot Token 正确、Bot 已发过 /start，再看 openclaw status 和网关日志。官方文档：Telegram 渠道

🎮 Discord Bot 配置

适合 Discord 用户，支持服务器和私信。

创建 Discord 应用

  1. 访问 Discord Developer Portal
  2. 点击 "New Application"，输入名称
  3. 左侧菜单选择 "Bot"，点击 "Add Bot"
  4. 复制 Bot Token（点击 "Reset Token" 生成新 Token）

获取 OAuth URL

  1. 左侧菜单选择 "OAuth2" → "URL Generator"
  2. Scopes 勾选 bot 和 applications.commands
  3. Bot Permissions 勾选：
    • Send Messages
    • Read Message History
    • Use Slash Commands
  4. 复制生成的 URL，在浏览器打开，邀请 Bot 到服务器

配置 OpenClaw

~/.openclaw/openclaw.json
{ "channels": { "discord": { "enabled": true, "token": "你的 Bot Token", "dmPolicy": "pairing", "groupPolicy": "allowlist", "guilds": { "你的 Server ID": { "requireMention": true, "users": ["你的 User ID"] } } } } }
💡
Slash 命令
Discord 支持 Slash 命令，输入 / 可以看到所有可用命令。
🧩
Discord 常见问题排查
Bot 在线但不回：重点检查 Bot 是否有目标频道发言权限、消息内容读取权限、以及 OAuth scopes 是否包含 applications.commands。官方文档：Discord 渠道

💬 WhatsApp 配置

使用你的个人 WhatsApp 账号接入。

运行配置向导

命令行
openclaw onboard

选择 WhatsApp，向导会显示一个二维码。

扫码登录

  1. 打开手机 WhatsApp
  2. 设置 → 已关联的设备 → 关联设备
  3. 扫描终端显示的二维码

开始使用

扫码成功后，你会收到一条确认消息。之后可以：

  • 给自己发消息与 AI 对话
  • 在群聊中 @你的 WhatsApp 号码调用 AI
⚠️
注意事项
WhatsApp 个人号接入使用了非官方客户端，有极小概率被限制。建议使用专门的小号，或考虑 Telegram Bot 方案。
🛠️
WhatsApp 常见问题排查
二维码反复失效：先退出旧会话并重启 Gateway，再重新扫码；如果云服务器无图形界面，建议用 Telegram/Discord 先跑通。官方文档：WhatsApp 渠道

🍎 iMessage 配置

macOS 专属，原生体验。

💡
系统要求
仅支持 macOS 12+ (Monterey) 及以上版本。需要登录 iCloud 账号并开启 iMessage。

授权 iMessage

终端
openclaw onboard

选择 iMessage，系统会弹出授权请求，点击"允许"。

开始使用

配置完成后，你可以：

  • 给自己的 Apple ID 发消息与 AI 对话
  • 在群聊中提及 AI 触发对话
✨
隐私优先
iMessage 消息使用端到端加密，Apple 无法读取。OpenClaw 在本地处理消息，不会上传到任何第三方服务器。
🧭
iMessage 常见问题排查
无法触发消息：确认 macOS 已登录同一 Apple ID、iMessage 可正常收发，并给 OpenClaw 授予自动化/辅助功能权限。官方文档：iMessage 渠道

🔒 Signal 配置

注重隐私的加密通讯方案。

安装 Signal CLI

Signal 需要通过 Signal CLI 接入。请参考 signal-cli 官方文档 安装。

注册或关联号码

建议使用专用号码注册 Signal CLI，而不是关联主号。

bash
signal-cli -u +86XXXXXXXXXXX register

配置 OpenClaw

~/.openclaw/openclaw.json
{ "channels": { "signal": { "enabled": true, "phoneNumber": "+86XXXXXXXXXXX" } } }
🧪
Signal 常见问题排查
注册失败通常是号码或验证码流程问题；建议先独立确认 signal-cli 可收发，再接入 OpenClaw。官方文档：Signal 渠道

💼 Slack 配置

企业团队协作场景。

创建 Slack App

  1. 访问 Slack API: Applications
  2. 点击 "Create New App" → "From scratch"
  3. 输入 App 名称，选择工作区

配置权限

  1. 左侧选择 "OAuth & Permissions"
  2. 在 "Bot Token Scopes" 添加：
    • app_mentions:read
    • chat:write
    • channels:history
    • groups:history
  3. 点击 "Install to Workspace" 并授权
  4. 复制生成的 Bot User OAuth Token

配置 OpenClaw

~/.openclaw/openclaw.json
{ "channels": { "slack": { "enabled": true, "botToken": "xoxb-xxxxxxxxxxxx-xxxxxxxxxxxx-xxxxxxxxxxxxxxxxxxxxxxxx", "appToken": "xapp-xxxxxxxxxxxx-xxxxxxxxxxxx-xxxxxxxxxxxxxxxxxxxxxxxx", "signingSecret": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" } } }

启用 Socket Mode

Slack 需要使用 Socket Mode 来接收事件：

  1. 左侧选择 "Socket Mode" → Enable
  2. 生成 App-Level Token（scope: connections:write）
  3. 将 App-Level Token 填入配置中的 appToken
🛠️
Slack 常见问题排查
大多数问题来自 Token/Scope 配置不全：确认 bot token、app token、signing secret 都已填，并已启用 Socket Mode。官方文档：Slack 渠道

🔌 同时使用多个渠道

OpenClaw 支持同时启用多个渠道。

~/.openclaw/openclaw.json
{ "channels": { "telegram": { "enabled": true, "token": "xxx" }, "discord": { "enabled": true, "token": "xxx", "clientId": "xxx" }, "whatsapp": { "enabled": false } } }

只要 enabled: true，Gateway 启动后会自动连接所有已启用的渠道。
